# 板材切割系统前端

本目录是前后端分离后的独立网页端工程，技术栈为 Vue 3、Vite、Pinia、Vue Router 和 Axios。

## 本地开发

```bash
npm install
npm run dev
```

开发环境默认通过 `/api` 代理到 `http://localhost:8080`，可在 `.env.development` 中修改 `VITE_BACKEND_URL`。

## 生产构建

```bash
npm run build
```

构建产物输出到 `dist/`，可由 Nginx 托管。生产环境接口同样使用 `/api` 前缀，由 Nginx 反向代理到后端服务。

## 页面范围

- 登录页
- 工作台概览
- 客户管理
- 板材管理
- 算法排样输入与结果可视化

## 与毕设文档的对应关系

当前网页端已覆盖毕设文档中的第一版后台基础能力：

- 登录与 token 管理
- 客户基础资料维护
- 板材基础资料维护
- 算法参数录入
- 排样结果可视化

暂未纳入当前网页端第一版的能力：

- 余料库
- 新建排单
- 历史排单
- 排样方案保存
- Excel 导入
- 刀轨 `.nc` 输出
- 标签打印
- 用户/角色管理

图片识别功能按产品定位由小程序端承载，网页端不实现拍照识别。
